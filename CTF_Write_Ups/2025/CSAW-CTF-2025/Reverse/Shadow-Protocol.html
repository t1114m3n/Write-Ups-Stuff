This program‚Äôs logic is based in a few functions :

  • main

This is the function orchestrating the whole program. After reading it carefully I understand the following :

  1. It uses the current time to seed a Pseudo-Random Number Generator (PRNG)
  1. Calls rand() twice to get two 32-bit words and combines it into a 64-bit seed
  1. Creates a binary tree seeded by this value

A binary tree is a data structure where each node can have up to two children:

  • left child
  • right child At the very top is the root node
  • Squares are internal node (with children)
  • Circle are leaves (no children)

  1. Calls shadow_tree_mix(tree, &local_f0, &local_f4) function to walk the tree and produce local_f0 (accumulator: collects a sequence of 3-bit chunks from each leaf) and local_f4 (leaf count: tells you how many leaves contributed to that sequence.)

  2. free_bittree(tree) ‚Üí free memory.

  3. local_c8 = shadow_protocol(local_f0) ‚Üí turn the accumulator into an 8-byte keystream mask.

  4. Read flag.txt and XOR each flag byte with the corresponding byte of local_c8(byte index modulo 8) and print hex

Once we know the principal logic of this program we can try to understand the two functions in charge of creating our XOR key:

  • shadow_tree_mix

From this function the only thing we need to know is that it will walk the binary tree and each time it hits a leaf, it will take it‚Äôs 3 last bits and append them into a big number, while counting the number of leaves seen.

  • And finally shadow_protocol function:

From this function we need to understand that it will take the number gotten from the tree and scramble it through substitutions, shifts and mixes, which will turn it into an 8-byte key.

All this information adds up to :

  1. We have the actual flag in flag.txt
  2. The flag is being XORed with an 8-byte key
  3. This key is derived from the actual time of the script‚Äôs execution

This means that based on the basic principles of XOR, we could just re-run the program with the encrypted key as input, and using the time given from the output of the program. Which will give us the flag in plaintext
