In order to bypass it, first we will need to upload frida server into the android device. It is important to note that the version of frida on your pc and the version of frida server on the emulated device must be the same, and must align with the architecture of the device:

  1. First verify the installed version of Frida
  1. Verify the architecture of the emulated phone

i686 corresponds to x86 system (32-Bit) so I will need to install a server for this architecture, If the response had been x86_64 it would mean that the system is a x64 (64-bit). In order to get a valid frida server here is the official repo : https://github.com/frida/frida/releases
