**Description**

I made this weird android app, but all it gave me was this .apk file. Can you get the flag from it?

This was a simple apk reversing challenge.

After decompressing the apk with :

  • https://www.decompiler.com/
  • https://apktool.org/

I started searching for known strings. From the flag format I know that ictf is the beginning of the flag so I start there.

I do a rip grep with ictf which gives me nothing. So I decide that it would be a good idea to open the app and see what it does.

I use : https://appetize.io/

So now I decide to search the following string : Transformed flag

Giving me the path to the MainActivityKt.java:

The script does the following:

There is a function named transformFlag which : For each character at position i in the input :

  1. If it’s a lowercase letter it shifts forward by i(mod 26)

res += alphabet[(alphabet_index + i) % 26]

  1. If it’s a digit it shifts it forward by 2*1(mod 10)

res += digits[(digit_index + 2*i) % 10]

  1. If it’s one of these symbols:

S = "!@#$%^&*()_+{}[]|"

It shifts it forward by i*i(mod |S| = 19)

res += S[(S_index + i*i) % len(S)]

  1. If the character doesn’t belong to any set it does nothing
